Cookbook

Copy-pasteable recipes for common tasks. Minimal explanation, maximum code.

137 words·1 min read

Recipes are short, complete, and runnable. Each one solves exactly one problem. For background on why something works the way it does, see the main docs.

💡

Every recipe uses the JSONPlaceholder API as the example. Swap in your own endpoints and the patterns transfer directly.

How recipes are structured

Each recipe has:

  1. The goal — one sentence
  2. The code — complete, copy-pasteable
  3. What to change — the two or three lines you'll need to adapt

Nothing else. If you want context, the main docs have it.
