What is Diátaxis?

Diátaxis is a framework that organises documentation into four types: Tutorials, How-to Guides, Reference, and Explanation. Each type serves a different user need.

Do I need to use all four types?

No. All four Diátaxis sections (and the special types like Cookbook) are opt-in. Set enabled: false in pepa.config.ts to hide any section you don't need.
