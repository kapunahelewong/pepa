## Syntax-highlighted code

```typescript
async function fetchWidgets(apiKey: string) {
  const res = await fetch("https://api.example.com/v1/widgets", {
    headers: { Authorization: `Bearer ${apiKey}` },
  });
  return res.json();
}
```

## Keyboard shortcuts

Press <Kbd>⌘K</Kbd> to open search.
Press <Kbd>⌘</Kbd>+<Kbd>Shift</Kbd>+<Kbd>P</Kbd> for the command palette.
