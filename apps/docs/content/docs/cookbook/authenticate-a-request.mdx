**Goal:** Make an authenticated request to the Widgets API.

## The code

```bash
curl https://api.example.com/v1/widgets \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```js
const res = await fetch("https://api.example.com/v1/widgets", {
  headers: {
    Authorization: `Bearer ${process.env.API_KEY}`,
  },
});
const { data } = await res.json();
```

```python
import os, httpx

resp = httpx.get(
    "https://api.example.com/v1/widgets",
    headers={"Authorization": f"Bearer {os.environ['API_KEY']}"},
)
resp.raise_for_status()
data = resp.json()["data"]
```

## What to change

- Replace `YOUR_API_KEY` / `process.env.API_KEY` / `os.environ['API_KEY']`
  with however your app loads secrets.
- Replace the URL with your real API base URL.
